Excerpt — FAQ:

What is Microsoft 365 E7?
Microsoft 365 E7 ("the Frontier Suite") is a new license tier announced
for May 2026. It unifies M365 E5, Microsoft 365 Copilot, and Agent 365
into a single bundle powered by Work IQ. It includes Microsoft Entra Suite
and advanced Defender, Intune, and Purview capabilities — sitting above E5
in the M365 commercial stack.

What is Microsoft Agent 365?
Microsoft Agent 365 is the control plane for AI agents, going general
availability in May 2026. It gives IT and security leaders a single place
to observe, govern, manage, and secure agents across the organization —
including agents built with Microsoft AI platforms and agents from
ecosystem partners.
